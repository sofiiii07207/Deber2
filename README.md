Descripción del ejercicio:
Este programa en C++ permite realizar conversiones de divisas. El usuario ingresa una divisa base, una divisa cotizada, la tasa de cambio y un monto base. El programa calcula el monto convertido y muestra un reporte con la información.

Lógica del programa:
1. Se solicitan las divisas base y cotizada.
2. Se valida que tengan exactamente 3 letras.
3. Se solicita la tasa de cambio y se valida que sea mayor a cero.
4. Se solicita el monto base y se valida que sea mayor a cero.
5. Se solicita la fecha y hora.
6. Se calcula el monto convertido multiplicando el monto base por la tasa.
7. Se muestra un reporte con todos los datos.
8. El usuario puede decidir si desea realizar otra conversión.
